ApiFlix

This app's objective is to build the server-side component of a “movies” web application.
The web application will provide users with access to information about different
movies, directors, and genres. Users will be able to sign up, update their
personal information, and create a list of their favorite movies.

1 >Return a list of ALL movies to the user /movies GET None None A JSON object holding data about all movies

3 Return data about a genre /movies/genre/:genreName GET :genreName None A JSON object holding data about the selected genre. Example: 
                
                  {
                    "Name": "Drama"
                  }
4 Return data about a director (bio) by name /movies/directors/:name GET :name None A JSON object holding data about the selected director. Example: 
                
                  {
                    "Name": "Paul McGuigan",
                    "Bio": "Paul McGuigan was born on September 19, 1963 in 
                            Bellshill, Scotland. He is a director and producer, 
                            known for Lucky Number Slevin (2006), Wicker Park 
                            (2004) and Victor Frankenstein (2015). He is married 
                            to Natasha Noramly. They have one child. He was 
                            previously married to Elisabeth McGuigan."
                  }
5 Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
                
                  {
                    "name": "Tina",
                    "favoriteMovies":[]
                
                  }
A JSON object holding data about added user including ID. Example: 
                
                  {
                    "name": "Tina",
                    "favoriteMovies": [],
                    "id": "08e51d15-14ba-4edd-bfdd-4d88646a4582"
                  }
6 Allow users to update their user info (username) /users/:id PUT :id A JSON object holding the new username. Example: 
                
                  {
                    "name": "Babe Ruth",
                    "favoriteMovies":[]
                  }
A JSON object with the updated user profile. Example: 
                
                  {
                    "id": 1,
                    "name": "Babe Ruth",
                    "favoriteMovies": [
                        "Rush Hour 2"
                    ]
                  }
8 Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST :id, :movieTitle None A text message indicating the movie was added
9 Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE :id, :movieTitle None A text message indicating the movie was deleted
10 Allow existing users to delete account /users/:id DELETE :id None A message indicating the user was deleted
